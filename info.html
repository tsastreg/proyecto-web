Información del Proyecto

Conoce más sobre el desarrollo de este sitio web del mejor gimnasio de Mallorca.

Sobre el Proyecto

Esta página forma parte de un trabajo del colegio donde estamos aprendiendo a montar una web paso a paso. Básicamente es practicar como poner textos, imágenes y secciones sin complicarnos demasiado.

La idea es hacer algo claro y facil de entender, como si fuera una mini web real del gimnasio, pero sin liarnos con cosas técnicas. Solo aprender lo básico mientras lo hacemos práctico.

Objetivos del Sitio

  • Dar información sobre las áreas del gimnasio.
  • Mostrar contenido visual representativo.
  • Facilitar la navegación mediante una interfaz sencilla.
  • Practicar con elementos de desarrollo web.

Información Adicional

Todo lo que veis aqui es solo para practicar, no es un gimnasio real. Las fotos y los textos estan puestos para que la web se vea un poco más completa mientras aprendemos a montarla.
